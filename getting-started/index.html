Create Your First PBR Material Switch Menu

We will demonstrate the powerful capabilities of EZMenu with a simple demo. Before we begin, let’s assume the following:

  • The model has already been uploaded to Second Life.
  • The material textures are ready.
  • EZMenu has been installed.

Let’s assume I have a model of a table and I’ve prepared three different wood textures: Wood1, Wood2, and Wood3. I want to switch between these materials using a menu.

Configure the Main Menu

When you first run EZMenu, it will automatically create a main menu and a permission control menu.

Add Menu Items

Under the Main Menu, we can create menu items: Wood1, Wood2, and Wood3. Additionally, add a permission control menu item Access so that the owner can set who can use the menu.

Link the Access menu item with the permission control menu item to implement access control.

Add Nodes

We can add the Set PBR Material node and link it with the Menu Item.

Right-click in an empty area, select Add Node, choose pbr, and then select Set PBR Material.

Left-click the Set PBR Material node. In this node, configure parameters including LinkNumber, Face, and Material.

For PBR Material setup, each Face must correspond to a Material. If the model has multiple Faces, a separate Set PBR Material node is required for each one.

In my model, there are 4 Faces, so I added 4 Set PBR Material nodes, each configured with a Face and a corresponding Material.

The setting for LinkNumber depends on specific needs. In my case, I set it to 0.

Prepare Materials

In Second Life, create the Textures you uploaded into Materials. If you don’t know how to do this, check out my following demo. If you’re already familiar with this process, you can skip this step.

Configure Faces and Materials

Once the materials are prepared, configure the corresponding Face and Material relationships in the Set PBR Material node.

Now, the Wood1 setup is complete. Link the Wood1 menu item to the four Set PBR Material nodes. When you click the Wood1 menu item, it will switch to the Wood1 material.

Repeat the same steps to configure and link the Set PBR Material nodes for the Wood2 and Wood3 menu items.

Generate the Script

Click Generate Script to generate the script. Copy the generated script into the script editor in Second Life and click Save.

Also, add the core script to the model. This completes the configuration of your menu. You’ve now successfully created a PBR material switch menu.

Summary

With these simple steps, we’ve created a PBR material switch menu. Isn’t it easy?
