常见问题

我的产品有了其他功能的菜单，还可以使用 EZMenu 切换材质吗？

可以， EZMenu 可以与其他功能共存，您可以自定义材质切换菜单的打开方式，比如通过link消息或者listen消息。

我需要更换材质的贴图，但不想更改整个材质的设置，应该如何操作？

使用 Set PBR Texture 节点，选择要更换的贴图类型（如 Base Color、Normal、(O)RM、Emissive），并提供新的贴图 UUID。这将只更换指定的贴图，而不会影响其他材质参数。

我需要更换材质的贴图，并且还需要调整贴图的缩放、偏移和旋转，应该如何操作？

使用 Set PBR Texture Adv 节点，选择要更换的贴图类型（如 Base Color、Normal、(O)RM、Emissive），并提供新的贴图 UUID。此外，您还可以调整贴图的缩放、偏移和旋转参数。

权限控制的菜单项我不想放在主菜单，可以将其放在子菜单中吗？

可以。将子菜单的菜单项连接到权限控制的菜单即可。

打开权限控制菜单的菜单项可以自定义菜单项名字吗？

可以。在 MenuItem 属性中输入您想要的菜单项名称即可。

如何删除不需要的菜单项？

将不需要的菜单项文字删除掉即可删除菜单项。菜单项对应的链接也会被删除。
