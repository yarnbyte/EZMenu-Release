Set PBR Roughness

节点介绍

该节点用于设置模型材质 Metallic-Roughness 贴图的粗糙度。

输入

  • MenuItem - 菜单项。

输出

  • 无

节点属性

  • PrimTag -

    该物件的唯一标识为PrimTag。 当模型由多个prim链接组成时，每个prim的描述中必须包含格式为 ez:xxxxx;的PrimTag（以ez:开头并以 ;结尾），以便脚本准确定位需要操作的prim。 对于单个prim的模型，此字段可留空。

  • Face - 指定的面。 从下拉列表中选择一个面。
  • Roughness - 粗糙度。这个参数用来设置粗糙度的值。值的范围是0.0到1.0。
